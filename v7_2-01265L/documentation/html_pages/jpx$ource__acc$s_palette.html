jpx_codestream_source::access_palette

Java: Jpx_codestream_source.Access_palette

jp2_palette access_palette( )

Java: public native Jp2_palette Access_palette( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns an object which may be used to access any information recorded in a "Palette" box found within the codestream header (chdr) box or a relevant "Palette" box found within the default JP2 heade box. If there is none, the object's jp2_palette::get_num_luts function will return 0.
