::kdu_read_float

static float kdu_read_float( kdu_byte * & bp, kdu_byte * end)

[Declared in "../coresys/common/kdu_utils.h"]

Synopsis

Reads a 4-byte single-precision floating point quantity from the array identified by bp. During the process, bp is advanced 4 bytes. If this pushes it up to or past the end pointer, the function throws an exception of type kdu_byte *.

The byte order is assumed to be big-endian. If the local machine architecture is little-endian, the input bytes are reversed.

Return Value

The value of the floating point quantity recovered from the first 4 bytes of the buffer.

Arguments

bp [kdu_byte * &]

Pointer to the first byte in the buffer from which the integer is to be recovered.

end [kdu_byte *]

Points immediately beyond the last valid entry in the buffer.
