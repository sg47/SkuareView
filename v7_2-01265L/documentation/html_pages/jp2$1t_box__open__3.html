Synopsis

This special version of the overloaded open function may be used to open a top-level box which has no associated jp2_family_tgt into which to dump its contents. The contents of the box will be buffered up as required, but never written. Before closing the box, however, you can always call get_contents to gain access to the box's internal buffer.
