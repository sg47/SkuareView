Synopsis

This function recursively examines the current node and all of its descendants, counting the total number of number list nodes that are encounted — i.e., nodes whose get_numlist_info function returns true. If value returned through count might potentially change in the future (because the ultimate source of data is a dynamic cache and not all of the information is yet available), the function returns false; otherwise it returns true.

Arguments

count [int &]
