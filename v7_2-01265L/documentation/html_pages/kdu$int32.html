kdu_interlocked_int32 [struct]

[Declared in "../coresys/common/kdu_elementary.h"]

Synopsis

This object has the size of a single 32-bit integer, but offers in-line functions to support atomic read-modify-write style operations for efficient inter-thread communication. All operations are accompanied by appropriate memory barriers, unless otherwise noted.
