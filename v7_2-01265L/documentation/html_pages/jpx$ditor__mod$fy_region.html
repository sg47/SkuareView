Synopsis

You can use this method to explicitly update an existing region by passing in new values for it. The src object is copied and the function returns true, unless idx lies outside the range 0 to R-1, where R is the number of regions returned via get_regions, in which case the function returns false.

idx [int]

src [const jpx_roi &]
