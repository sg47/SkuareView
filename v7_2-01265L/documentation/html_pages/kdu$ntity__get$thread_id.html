kdu_thread_entity::get_thread_id

Java: Kdu_thread_entity.Get_thread_id

int get_thread_id( )

Java: public native int Get_thread_id( )

[Declared in "../coresys/common/kdu_threads.h"]

Go to class description.

Synopsis

Returns the identity of this kdu_thread_entity object. The return value will be 0 if this is the group owner (see is_group_owner) or the create function has not yet been called. Otherwise, the object refers to a worker thread and will have an identity greater than 0. The returned identity values lie in the range 0 to N-1, where N is the number of threads in the thread group — see get_num_threads.
