[Declared in "../apps/support/kdu_region_compositor.h"]

Synopsis

Returns the number of auxiliary painting parameters supplied via the most recent call to kdu_region_compositor::configure_overlays. If you attempt to access more than this number of parameters using the get_aux_param function, you will be returned 0.
