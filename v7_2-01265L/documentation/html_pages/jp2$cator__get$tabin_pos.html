jp2_locator::get_databin_pos

Java: Jp2_locator.Get_databin_pos

kdu_long get_databin_pos( )

Java: public native long Get_databin_pos( )

[Declared in "../apps/compressed_io/jp2.h"]

Go to class description.

Synopsis

Returns the location associated with this locator, relative to the start of the databin returned by get_databin_id. If there is no databin, the value returned by this function is meaningless but should generally be 0.
