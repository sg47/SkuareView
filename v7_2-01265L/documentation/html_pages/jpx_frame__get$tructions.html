jpx_frame::get_num_persistent_instructions

Java: public native int Get_num_persistent_instructions( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns the number of initial instructions recognized by this interface that correspond to previous persistent animation frames. To inspect only the unique instructions for this particular frame, call get_instruction with instruction indices that are greater than or equal to the value returned by this function. The return value from this function will be zero if the interface was created without the include_persistents option — see jpx_composition::access_frame, for example.
