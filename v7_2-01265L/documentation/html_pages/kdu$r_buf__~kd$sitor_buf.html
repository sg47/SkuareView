virtual ~kdu_compositor_buf( )

Java: public native Native_destroy( )

[Declared in "../apps/support/kdu_region_compositor.h"]

Go to class description.

Synopsis

If you are providing your own buffers, based on a derived version of this class, beware that the internal buf and float_buf arrays will be passed to the C++ delete operator if non-NULL. If you have allocated these using anything other than new, be sure to override the destructor and set them to NULL before exiting your version of the function.
