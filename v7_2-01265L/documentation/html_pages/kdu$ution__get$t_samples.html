Synopsis

Returns the total number of code-block samples which belong to the precinct identified by precinct_idx. This can be 0 if the precinct contains no code-blocks — some precincts in JPEG2000 can contain no code-blocks yet exist nonetheless, depending upon the image dimensions.

The precinct_idx argument should lie within the range of indices returned via get_valid_precincts.

Arguments

precinct_idx [kdu_coords]
