jpx_target::operator=

jpx_target & operator=( jpx_target & rhs)

[Declared in "../apps/compressed_io/jpx.h"]

Synopsis

This assignment operator serves to prevent actual copying of one jpx_target object to another. In debug mode, it raises an assertion to help you catch such mistakes.
