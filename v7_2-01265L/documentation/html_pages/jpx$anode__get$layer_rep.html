Synopsis

If this metanode is not embedded within a JPX container (i.e., if get_container_id returns -1), or if layer_idx is not one of the compositing layers defined by a container in which the metanode is developed, this function returns 0.

Otherwise, the function returns the repetition instance of the JPX container to which layer_idx belongs. Mostly this function would be applied to number list nodes, but the function works on any type of metanode.

The main purpose of the function is to walk through the codestream indices defined by a number list that belong to the same repetition instance of the container (if any) as a compositing layer index — this can be done by passing the returned repetition index as the second argument to get_numlist_codestream.

layer_idx [int]
