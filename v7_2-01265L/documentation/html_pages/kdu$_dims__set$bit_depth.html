kdu_image_dims::set_bit_depth

void set_bit_depth( int comp_idx, int bit_depth)

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

An image file reading module can use this function to change the bit-depth of an image component which has already been added, using this function. This is mainly provided to simplify the implementation of precision forcing, as described in connection with the get_forced_precision function. The signed/unsigned attributes are unaffected by this function. The function expects the component to exist already.

Arguments

comp_idx [int]

bit_depth [int]
