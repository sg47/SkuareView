jp2_output_box::get_box_length

[Declared in "../apps/compressed_io/jp2.h"]

Go to class description.

Synopsis

Returns the number of bytes in the box, including its header (unless the box was opened as header-less). If this function is called before all bytes in the box have been written, the header contribution (unless header-less) may be inaccurately calculated, since boxes with size greater than or equal to 2^32 require 16-byte headers instead of 8-byte headers. Usually, you will wait until you are about to close the box (or even after you have closed the box) before calling this function, although the size of a header can be guaranteed ahead of time by invoking the use_long_header or set_target_size function.
