kdu_args

[Declared in "../apps/args/kdu_args.h"]

Synopsis

This object provides a convenient set of utilities for digesting and processing command-line arguments. Its convenient features include the ability to search for an argument, automatic inclusion of arguments from switch files, and detection and reporting of unused arguments.
