kdu_roi_level::kdu_roi_level

kdu_roi_level( )

[Declared in "../coresys/common/kdu_roi_processing.h"]

Go to class description.

Synopsis

Creates an empty interface; one whose exists function returns false. You must call the create function to construct the internal machinery to which the interface refers. Only then can you use the acquire_node function.
