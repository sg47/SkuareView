Synopsis

Almost the same as calling clear, except that the function also sets the object's stateless attribute. Use this function if you wish to return the object to an empty state, correctly reporting its statelessness via is_stateless, but you do not yet need to add any cache model instructions. Use the second form of init if you need to add cache model instructions. Either of the init functions may be called as often as you like.

Arguments

stateless [bool]
