kdu_image_dims::get_colour_info

int get_colour_info( bool & has_premultiplied_alpha, bool & has_unassociated_alpha, int & colour_space_confidence, jp2_colour_space & colour_space, int & profile_len)

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

This function may be used to retrieve any knowledge which might be available concerning the colour space and/or presence of alpha components.

Return Value

0 if nothing is known about the colour or alpha properties of the image component samples. Otherwise, the returned value represents the number of colour channels.

Arguments

has_premultiplied_alpha [bool &]

Used to return an indication of whether or not there is a pre-multiplied alpha channel. See set_colour_info for an explanation.

has_unassociated_alpha [bool &]

Used to return an indication of whether or not there is an unassociated alpha channel. See set_colour_info for an explanation.

colour_space_confidence [int &]

Used to return the colour space confidence. See set_colour_info for an explanation.

colour_space [jp2_colour_space &]

The value returned here is meaningful only if the colour_space_confidence value is greater than 0.

profile_len [int &]

If the value returned value is positive, the colour space is described by an ICC profile, that may subsequently be retrieved using the get_profile function. The first profile_len bytes of the returned buffer represents the complete ICC profile.
