jpx_target::access_container

Java: Jpx_target.Access_container

jpx_container_target access_container( int which)

Java: public native Jpx_container_target Access_container( int which)

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

This function may be used to recover an interface to a JPX container that has previously been added using add_container. If which is negative or is greater than or equal to the number of preceding calls to add_container, an empty interface is returned.

Arguments

which [int]
