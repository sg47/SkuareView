[Declared in "../coresys/common/kdu_roi_processing.h"]

Go to class description.

Synopsis

This function is provided for backward compatibility with Kakadu versions 4.5 and earlier, where each stage in the DWT decomposition tree was necessarily an image resolution. This version of the overloaded create function just invokes the first version with the resolution's primary node, obtained via its kdu_resolution::access_node function.

Arguments

resolution [kdu_resolution]

source [kdu_roi_node *]
