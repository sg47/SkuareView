jp2_input_box::get_codestream_scope

Java: Jp2_input_box.Get_codestream_scope

kdu_long get_codestream_scope( )

Java: public native long Get_codestream_scope( )

[Declared in "../apps/compressed_io/jp2.h"]

Go to class description.

Synopsis

Returns -ve unless the ultimate data source is a kdu_cache and the current object refers to an open incremental codestream placeholder box. In that case, the function returns the logical codestream that the object is currently configured to access within the dynamic cache — may be changed via set_codestream_scope. The logical codestream index has a well defined meaning for each type of source file, but wherever the source offers a natural one-dimensional index for codestreams that index should correspond to the logical codestream index.
