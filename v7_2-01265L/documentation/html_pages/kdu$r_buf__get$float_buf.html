Synopsis

Same as get_buf, except that this function is used to retrieve buffers which have a floating point organization, as instantiated by the init_float function.

Arguments

row_gap [int &]

For this function, row_gap returns the separation between successive rows, measured in samples (not pixels) — each sample is a single floating point number.

read_write [bool]
