kdu_node::get_kernel_id

Java: Kdu_node.Get_kernel_id

int get_kernel_id( )

Java: public native int Get_kernel_id( )

[Declared in "../coresys/common/kdu_compressed.h"]

Go to class description.

Synopsis

Returns one of Ckernels_W5X3, Ckernels_W9X7 or Ckernels_ATK (-1). In the former two cases, the return value may be passed directly to kdu_kernels::init. In the last case, however, kernel information is stored in an ATK marker segment. In view of this, the only general purpose way to recover transform kernel information is to call get_kernel_info and get_kernel_coefficients.
