kdu_args::get_prog_name

char * get_prog_name( )

[Declared in "../apps/args/kdu_args.h"]

Go to class description.

Synopsis

Returns a pointer to an internal copy of the first string in the argv array supplied to the constructor. As explained in the description of kdu_args::kdu_args, this is expected to be the name of the program.
