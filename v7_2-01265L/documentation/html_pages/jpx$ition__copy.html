jpx_composition::copy

Java: Jpx_composition.Copy

void copy( jpx_composition src)

Java: public native void Copy( Jpx_composition src)

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Use this function to copy the composition instructions from the src object to the present object. This works both for top-level jpx_composition object and those that are embedded within JPX containers — see jpx_container_source::access_presentation_track and jpx_container_target::add_presentation_track.

Arguments

src [jpx_composition]
