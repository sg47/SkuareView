kdu_client::is_one_time_request

Java: Kdu_client.Is_one_time_request

bool is_one_time_request( )

Java: public native boolean Is_one_time_request( )

[Declared in "../apps/compressed_io/kdu_client.h"]

Go to class description.

Synopsis

Returns true if the client is operating in the non-interactive mode, as determined by the mode argument passed to connect. The return value from this function remains the same from the call to connect until close is called (potentially from inside a subsequent connect call).
