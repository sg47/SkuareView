Synopsis

Returns true if the image component originally had a signed representation; false if it was unsigned. The interpretation of the component index is affected by calls to apply_input_restrictions as well as the want_output_comps argument. This is explained in connection with the get_bit_depth function.

Arguments

comp_idx [int]

want_output_comps [bool]

Affects the interpretation of comp_idx, as explained in connection with get_bit_depth. Note that this argument is relevant only if if there have been no calls to apply_input_restrictions or the most recent call specified KDU_WANT_OUTPUT_COMPONENTS.
