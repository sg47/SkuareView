jpx_composited_orientation::append

Java: Jpx_composited_orientation.Append

Java: public native void Append( Jpx_composited_orientation rhs)

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Appends the rhs re-orientation operation to any re-orientation operation associated with the current object, storing the overally re-orientation effect within the current object. That is, the re-orientation steps associated with the current object (on entry) are considered to be applied first, followed by those associated with rhs.

rhs [const jpx_composited_orientation &]
