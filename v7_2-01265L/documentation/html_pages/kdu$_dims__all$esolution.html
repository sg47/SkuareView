kdu_image_dims::allow_save_resolution

bool allow_save_resolution( )

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

This function always returns true except in the event that set_resolution has been called with its dont_save argument equal to true — in which case any resolution information recovered via get_resolution should not be saved in an output file.
