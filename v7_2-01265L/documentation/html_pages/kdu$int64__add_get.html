kdu_interlocked_int64::add_get

kdu_int64 add_get( kdu_int64 val)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Same as get_add, except that it returns the new value of the variable, after adding get. Again, there are no memory barriers and there is no guarantee of atomicity in the addition.

val [kdu_int64]
