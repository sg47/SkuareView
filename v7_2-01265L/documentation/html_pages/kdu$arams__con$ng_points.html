Synopsis

If you intend to use get_ring_points, call this function first with the stride associated with the buffer you intend to paint via your overlay painting routine and the radius with which you intend to draw rings. The function configures internal lookup tables so as to minimize computation in the painting of border rings — see get_ring_points for more info. Calls to this function need not necessarily do anything, since the object caches the ring lookup tables it generated in the past.

Arguments

stride [int]

radius [int]

This value should not exceed the value returned by get_max_painting_border, as ring lookup tables are generated only for radii in the range 0 to the maximum painting border.
