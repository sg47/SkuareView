kdu_semaphore::signal

bool signal( )

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Use this function to increment the semaphore by 1, waking up any thread that may have yielded execution in a call to wait unless this would reduce the semaphore's count below 0.

Return Value

False if the semaphore has not been successfully created.
