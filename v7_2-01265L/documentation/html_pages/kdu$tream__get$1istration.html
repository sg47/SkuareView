Synopsis

This function is almost identical to get_registration, except that registration offsets are measured relative to any registration offsets associated with the reference component, as identified by ref_comp_idx. If the reference image component has no registration offset, this function's behaviour is identical to get_registration. If the two image components both have the same registration offset, the function sets crg to (0,0).
