Synopsis

Returns true if the edge from vertex[n] to vertex[(n+1) mod 4] intersects the edge from v1 to v2. An intersection between edges A-B and C-D exists if there exists an internal point along the edge A-B (i.e., not equal to A or B) which is also an internal point along the edge C-D (i.e., not equal to C or D).

This function is invoked automatically from within check_geometry but you might find it useful all by itself, for locating offending edges explicitly.

Arguments

n [int]

v1 [kdu_coords]
