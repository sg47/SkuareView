jpx_roi_editor::is_empty

Java: Jpx_roi_editor.Is_empty

bool is_empty( )

Java: public native boolean Is_empty( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns true if the object contains no ROI description information at all. This state cannot be achieved by editing itself (e.g., by calls to delete_region) since the editing operations insist that at least one region be present. However, is_empty becomes true after a call to reset, or if init has never been called.
