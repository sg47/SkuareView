Synopsis

Use this function to set up a bandwidth slice preference, rather than explicitly setting the object's data members. The bw_slice member identifies a relative slice of the total available bandwidth to be assigned to a specific JPIP channel. As with set_pref, this function ensures that only one of preferred or required signals the preference. However, you are advised against making the preference required.

bw_slice [kdu_uint32]
