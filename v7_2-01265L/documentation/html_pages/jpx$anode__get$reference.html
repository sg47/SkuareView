Synopsis

This function may be used to access information for cross-reference nodes — any node derived from a cref box, having box-type jp2_cross_reference_4cc. If the node is not a cross-reference node, the function returns 0. Otherwise, the function returns the box_type of the referenced box and a fragment list identifying the (potentially fragmented) contents of the referenced box. These may be passed to jpx_input_box::open_as in order to open the referenced box.
