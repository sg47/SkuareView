kdu_interlocked_int64::exchange_or

kdu_int64 exchange_or( kdu_int64 val)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Performs an atomic logical OR of the variable with the supplied value, returning the original value (prior to the logical OR). The operation can be considered a full memory barrier.

Arguments

val [kdu_int64]
