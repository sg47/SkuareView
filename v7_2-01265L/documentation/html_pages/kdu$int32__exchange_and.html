kdu_interlocked_int32::exchange_and

kdu_int32 exchange_and( kdu_int32 val)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Atomically AND's val with the variable, returning the original value (prior to the logical AND). The operation can be considered to offer a full memory barrier.

Arguments

val [kdu_int32]
