jpx_codestream_source::get_header_loc

Java: Jpx_codestream_source.Get_header_loc

jp2_locator get_header_loc( )

Java: public native Jp2_locator Get_header_loc( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Use this function to obtain a jp2_locator object which describes the location of the codestream header (chdr) box for this codestream. If there is no codestream header box, the returned object's jp2_locator::is_null member will return true.
