kdu_tiffdir::get_fieldlength

Go to class description.

Synopsis

Returns the number of bytes per field (i.e., per tag value), based on the least significant 16 bits of the supplied tag_type. Essentially, this function just translates the data type values described in the introductory comments to this kdu_tiffdir object. The total number of bytes in the body of a TIFF tag may be found by multiplying this quantity by the value returned by get_taglength.

Note that the result of this function does not depend in any way upon the state of the object — in particular, the tag does not need to exist.

Return Value

0 if the tag_type has an unrecognized data type code. Otherwise, returns the number of bytes used for values of this type.

Arguments

tag_type [kdu_uint32]
