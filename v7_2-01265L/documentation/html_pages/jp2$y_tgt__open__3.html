Synopsis

This function opens a compressed target which does not actually write any data anywhere, but it does keep track of the last file position to which data was written, so that the get_bytes_written function returns the sum of the simulated_start_pos argument and the number of byte actually written by jp2_output_box objects which are opened with this target.

Arguments

simulated_start_pos [kdu_long]
