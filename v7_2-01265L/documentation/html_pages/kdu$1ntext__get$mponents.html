Synopsis

See kdu_serve_target::access_context for a general introduction. The remapping_ids and member_idx arguments have the same interpretation as in get_codestream, except that this function returns an array that lists all image components from the codestream which are used by this context.

Return Value

An array with one entry for each image component used by the indicated codestream context, from the specific codestream associated with the indicated context member. This is the same codestream whose index is returned get_codestream. Each element in the array holds the index (starting from 0) of the image component.

Arguments

remapping_ids[] [const int]

member_idx [int]

num_components [int &]

Used to return the number of elements in the returned array.
