kdu_args::advance

char * advance( bool remove_last=true)

[Declared in "../apps/args/kdu_args.h"]

Go to class description.

Synopsis

Advances to the next argument. If remove_last is true, the most recent argument returned via any of get_first, find or advance is deleted.

The function returns NULL if we try to advance past the last argument, or if the most recent call to get_first, find or advance returned NULL.

Arguments

remove_last [bool]
