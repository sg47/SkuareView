kdu_thread::set_to_self

bool set_to_self( )

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

You can use this function to create a reference to the caller's own thread. The resulting object can be passed to equals, get_priority, set_priority and set_cpu_affinity. It can also be passed to destroy, although this will not do anything.

If you want to obtain a reference to a kdu_thread object from within the thread that it was used to create, this can be done using the static get_thread_ref function.

Return Value

False if thread functionality is not supported on the current platform. This problem can usually be resolved by creating the appropriate definitions in "kdu_elementary.h".

The function also returns false if the object is the subject of a previous successful call to create and the matching destroy call has not yet been received.
