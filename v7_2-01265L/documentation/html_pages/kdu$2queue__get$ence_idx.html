kdu_thread_queue::get_sequence_idx

[Declared in "../coresys/common/kdu_threads.h"]

Go to class description.

Synopsis

Returns the sequence index used for jobs scheduled by this object's schedule_jobs function. This index is the larger of the min_sequence_idx value passed to add_thread when the queue was added and the sequence index of any parent ( super_queue) that was passed in the same call, or possibly larger again if some adjustments in the sequence index turned out to be required. See kdu_thread_entity::attach_queue for more on the interpretation of sequence indices. Knowing a queue's sequence index can be useful in helping you create other queues with the same or a larger sequence index.
