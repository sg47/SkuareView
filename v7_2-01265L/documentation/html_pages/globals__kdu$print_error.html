::kdu_print_error

Java: static Kdu_global.Kdu_print_error

void kdu_print_error( const char * message)

Java: public native static void Kdu_print_error( String message)

[Declared in "../coresys/common/kdu_messaging.h"]

Synopsis

This function provides a compact means of generating simple error messages through kdu_error. It is particularly useful for foreign language bindings which require all objects to reside on the heap, since kdu_error objects should never be created on the heap.

Arguments

message [const char *]
