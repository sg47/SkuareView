kdu_window_model::is_stateless

Java: Kdu_window_model.Is_stateless

bool is_stateless( )

Java: public native boolean Is_stateless( )

[Declared in "../apps/compressed_io/kdu_client_window.h"]

Go to class description.

Synopsis

Returns true if the cache model is to be used to process a stateless request. This means that all pre-existing cache model content is to be discarded and only those cache model instructions which relate to requested content need be considered.
