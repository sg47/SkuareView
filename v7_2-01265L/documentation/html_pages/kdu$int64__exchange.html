kdu_interlocked_int64::exchange

kdu_int64 exchange( kdu_int64 new_val)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Atomically writes new_val to the variable and returns the previous value of the variable. The operation can be considered to offer a full memory barrier.

Arguments

new_val [kdu_int64]
