jpx_roi_editor

Java: class Jpx_roi_editor

[Declared in "../apps/compressed_io/jpx.h"]

Synopsis

This class provides core facilities which can be used to implement an interactive editor for JPX ROI shapes. A single instance of this class can manage all the individual jpx_roi regions in a complete ROI description box.
