jpx_source::access_data_references

Java: Jpx_source.Access_data_references

jp2_data_references access_data_references( )

Java: public native Jp2_data_references Access_data_references( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

This function returns an interface which may be used to interpret the contents of fragment table and/or fragment list boxes. If the JPX data references box ( dtbl) cannot yet be read for some reason (e.g., insufficient data available so far from a dynamic caching data source), the function returns an empty interface (i.e., one whose jp2_data_references::exists function returns false).
