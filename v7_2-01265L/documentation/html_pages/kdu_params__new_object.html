kdu_params::new_object

Java: Kdu_params.New_object

virtual kdu_params * new_object( )

Java: public native Kdu_params New_object( )

[Declared in "../coresys/common/kdu_params.h"]

Go to class description.

Synopsis

Creates a new object of the same class as the present one. This function is implemented within each derived parameter class to simply invoke that class's constructor. It is used by new_instance and access_relation whenever a new object must be created within an existing cluster.
