Synopsis

Use this function to set up a global maximum bandwidth preference, rather than explicitly setting the object's data members. The max_bw member is expressed in Bits per Second. As with set_pref, this function ensures that only one of preferred or required signals the preference. However, you are advised against making the preference required.

Arguments

max_bw [kdu_long]

make_required [bool]
