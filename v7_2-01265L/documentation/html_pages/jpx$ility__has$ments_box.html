jpx_compatibility::has_reader_requirements_box

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns true if the file contains a reader-requirements box or a reader requirements box will be written. Strictly speaking, every JPX or JPX-compatible file must contains a reader requirements box, but early JPX files were sometimes written by other implementors without the reader requirements box. For this reason, we adopt a more lenient position here. If no reader requirements box is available, the check_standard_feature, check_vendor_feature and other related functions will all return false.
