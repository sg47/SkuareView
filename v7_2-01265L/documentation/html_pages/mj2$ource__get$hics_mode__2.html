[Declared in "../apps/compressed_io/mj2.h"]

Go to class description.

Synopsis

Same as first form of the overloaded get_graphics_mode function, except that no op-colour information is returned. If the return value is MJ2_GRAPHICS_BLUE_SCREEN, you will have to use the first form to obtain the op-colour for blue screening.
