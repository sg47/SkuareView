kdu_serve::retrieve_extra_data_chunks

kds_chunk * retrieve_extra_data_chunks( int context_id=0)

[Declared in "../apps/kdu_server/kdu_serve.h"]

Go to class description.

Synopsis

You may use this function to retrieve the internal list of chunks which has been assembled from calls to push_extra_data (with a NULL chunk_list argument), since the extra data was last returned via a call to this function or generate_increments.

Arguments

context_id [int]

As explained with push_extra_data, each window context has its own internal extra data queue. If no context with the supplied ID has yet been used, this function returns NULL immediately, without creating a new context.
