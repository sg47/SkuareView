::kdu_get_num_processors

Java: static Kdu_global.Kdu_get_num_processors

int kdu_get_num_processors( )

Java: public native static int Kdu_get_num_processors( )

[Declared in "../coresys/common/kdu_arch.h"]

Synopsis

This function returns the total number of logical processors which are available to the current process, or 0 if the value cannot be determined. For a variety of good reasons, POSIX refuses to standardize a consistent mechanism for discovering the number of system processors, although their are a variety of platform-specific methods around. If the number of processors cannot be discovered, this function returns 0.
