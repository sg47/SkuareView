kdu_thread::get_thread_ref

static kdu_thread * get_thread_ref( )

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

If kdu_thread::create is used to create a new thread of execution, that thread can later find a reference to its own kdu_thread object (the one that will eventually be used to to destroy the thread) by calling this static function.
