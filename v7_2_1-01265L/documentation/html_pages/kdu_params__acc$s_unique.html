Synopsis

Same as access_relation except that if the requested tile-component does not have its own unique object (i.e., if it simply references a default object with tile or component index = -1), the function returns NULL. This is particularly useful if you need to delete any objects which belong to a tile-component which is no longer in use.

comp_idx [int]
