kdu_window::init

Java: public native void Init( )

[Declared in "../apps/compressed_io/kdu_client_window.h"]

Synopsis

Sets all dimensions to zero. Sets the max_layers value to 0 (default value, meaning no restriction). Sets the collections of components and codestreams to empty. Sets round_direction to -1 (round-down is the default). Sets the metareq list to empty (NULL).
