kdu_clock::measures_real_time

bool measures_real_time( )

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Returns true if the object is supposed to measure ellapsed system time (real time). Every effort is made to ensure that this is true. Otherwise, it is possible (but not certain) that the measured time will be ellapsed CPU time.
