Synopsis

Returns true if the region of interest is a simple rectangle; otherwise, the function returns false and does not touch the rectangle argument. For interpretation of coordinates, see set_rectangle. You may wish to use get_quadrilateral instead of get_rectangle, since rectangles are a special case of quadrilaterals.
