Synopsis

Convenience function to write a single 64-bit tag value. See the fifth form of the write_tag function for more info. Note that this function may be used with tags whose data type (least significant two bytes of the tag_type) is any of 3, 8, 4, 9, 16 or 17. It is the preferred function to use for single-samle writing through foreign language bindings, where argument data-type alone might not be sufficient to resolve ambiguity.
