Synopsis

This is one of four functions which work together to translate codestream contexts into codestreams, and view windows expressed relative to a codestream context into view windows expressed relative to the individual codestreams associated with those contexts.

JPIP contexts are obtained by passing the context_idx, context_type and remapping_ids values extracted from a JPIP window of interest request to kdu_serve_target::access_context or kdu_client_translator::access_context.

Return Value

The number of codestreams which belong to the context.

Arguments

remapping_ids[] [const int]

Array containing two integers, whose values may alter the context membership and/or the way in which view windows are to be mapped onto each member codestream. For more information on the interpretation of remapping id values, see the comments appearing with kdu_sampled_range::remapping_ids and kdu_sampled_range::context_type.
