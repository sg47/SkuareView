Synopsis

Get the tightest bounding box which contains all regions, including the effect of any drag which may be in progress. Returns false, leaving bb unchanged, if is_empty woudl return true.

Arguments

bb [kdu_dims &]

include_scribble [bool]

If true, the scribble points added using add_scribble_point are also included in the bounding region.
