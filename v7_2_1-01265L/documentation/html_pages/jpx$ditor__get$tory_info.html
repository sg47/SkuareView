Synopsis

Returns information about the state of the internal undo queue.

Return Value

The maximum undo history length, as set by set_max_undo_history.

Arguments

available_undo_elts [int &]

Used to return the number of previous editing steps which can be undone by calls to undo — this is the current length of the undo queue.

can_redo [bool &]

Used to return an indication of whether a previously undone editing step can be redone by a call to redo.
