Synopsis

Use this function to inspect specific elements on the processed composition buffer queue. For a discussion of this queue, as well as thread-safe access procedures, see the discussion accompanying the push_composition_buffer function.

Return Value

NULL if the element identified by elt does not exist. If elt=0 and the function returns NULL, the composition buffer queue is empty. If the element identified by elt exists, the function returns a pointer to the relevant composition buffer.

By overriding the allocate_buffer function you can arrange for the buffers that are actually pushed onto the composition queue to be derived objects that carry additional state information.

Arguments

elt [int]

Identifies the particular queue element to be inspected. A value of 0 corresponds to the head of the queue. A value of N-1 corresponds to the tail of the queue, where the queue holds N elements.

custom_stamp [kdu_long *]

If non-NULL and the function returns true, this argument is used to return the stamp value which was supplied in the corresponding call to push_composition_buffer.

custom_id_val [int *]

If non-NULL and the function returns true, this argument is used to return the identifier which was supplied in the corresponding call to push_composition_buffer.
