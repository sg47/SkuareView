mj2_video_source::access_colour

Java: Mj2_video_source.Access_colour

jp2_colour access_colour( )

Java: public native Jp2_colour Access_colour( )

[Declared in "../apps/compressed_io/mj2.h"]

Go to class description.

Synopsis

Returns an object which may be used to access information recorded in the JP2 "Color" box, which indicates the interpretation of colour image data for rendering purposes. The returned jp2_colour object also provides convenient colour transformation functions, to convert data which uses a custom ICC profile into one of the standard rendering spaces. The colour information is fixed for all frames in the video track.
