kdu_region_compositor::get_max_available_quality_layers

Go to class description.

Synopsis

Returns the maximum number of quality layers which are available (even if not yet loaded into a dynamic cache) from any code-stream currently in use. This value is useful when determining a suitable value to be supplied to set_max_quality_layers.

Return Value

Returns 0 if there are no code-streams currently in use. Returns 1 if there is at least one code-stream in use, but no tiles of any code-streams have yet been opened — the number of layers is not actually known until tiles are opened.
