jp2_output_box::get_start_pos

Java: Jp2_output_box.Get_start_pos

kdu_long get_start_pos( )

Java: public native long Get_start_pos( )

[Declared in "../apps/compressed_io/jp2.h"]

Go to class description.

Synopsis

Returns the absolute location of the first byte in the box header (or the first byte in the box's body, if the box was opened as header-less), relative to the first byte written (or to be written) to the underlying jp2_family_tgt object.

Note that this function relies upon knowing the lengths of headers that might not yet have been written or allocated for super-boxes. For this reason, if this is a sub-box, you are generally advised either to ensure that no super-box will ever have a length of 2^32 bytes or more, or to first invoke use_long_header on all super-boxes.

As with open_next, this function relies upon any super_box that was passed to the open function continuing to exist. If this is a sub-box and the super-box may have been deallocated or corrupted, you must not invoke this function.
