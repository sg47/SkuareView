Synopsis

This function conveniently performs all the coordinate mapping operations prescribed by the custom_paint_overlay function, returning an array with num_regions entries, corresponding to transformed versions of the entries supplied with the regions array on input. The copies belong to an internal array managed by the present object. You may feel free to modify them further if required.
