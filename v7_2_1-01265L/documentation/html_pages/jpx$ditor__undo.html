jpx_roi_editor::undo

Java: Jpx_roi_editor.Undo

kdu_dims undo( )

Java: public native Kdu_dims Undo( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

You may call this function to undo the most recent completed editing operation, if any. If a drag operation is in progress (i.e., a call to drag_selected_anchor has not yet been followed by a call to move_selected_anchor or cancel_drag), the function simply invokes cancel_drag.

Return Value

A region large enough to include all points and edges affected by the change. If there is no change, this region is empty.
