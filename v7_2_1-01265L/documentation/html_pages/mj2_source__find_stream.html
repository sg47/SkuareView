Synopsis

Finds the track containing the codestream whose absolute index is supplied by stream_idx, returning the corresponding frame and field indices within that track via the frame_idx and field_idx arguments.

If the function is unable to determine whether the stream exists, the function returns false. This may happen if insufficient information is available from a dynamic cache. Otherwise, the function returns true, even if the stream is invalid. An invalid stream can be identified by the fact that the returned track_idx value will be 0.

Return Value

False if insufficient information exists to determine whether or not the stream exists. This condition may change in the future if more data is added to a dynamic cache. If the function returns true, the values of track_idx, frame_idx and field_idx will be set appropriately.

Arguments

stream_idx [int]

Absolute codestream index — see mj2_video_source::get_stream_idx for an explanation of this.

track_idx [kdu_uint32 &]

Used to return the index of the track to which the stream belongs. If the codestream index is invalid (i.e., it does not belong to any track), track_idx is set to 0.

frame_idx [int &]

Set to the index (starting from 0) of the frame to which the codestream belongs within its track — unaffected if the function returns 0.

field_idx [int &]

Set to the index (0 or 1) of the field to which the codestream belongs — unaffected if the function returns 0.
