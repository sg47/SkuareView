kdu_interlocked_int64::compare_and_set

bool compare_and_set( kdu_int64 ref_val, kdu_int64 set_val)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Atomically compares the variable with the supplied ref_val, returning false if the comparison fails (not equal), but returning true and setting the variable to set_val if the comparison succeeds (equal).

Arguments

ref_val [kdu_int64]

set_val [kdu_int64]
