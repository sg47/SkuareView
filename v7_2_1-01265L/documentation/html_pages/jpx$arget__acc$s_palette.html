[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Provides an interface which may be used for setting up a "Palette" (pclr) box.

It is NOT NECESSARY to access or initialize any palette information; the default behaviour is to not associate any palette with the codestream.
