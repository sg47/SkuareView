kdu_thread_entity_condition

Java: class Kdu_thread_entity_condition

[Declared in "../coresys/common/kdu_threads.h"]

Synopsis

References to objects of this class are used to wait upon conditions while doing work in the background — generally better than just waiting upon a synchronization object like kdu_event. The relevant functions are kdu_thread_entity::get_condition, kdu_thread_entity::wait_for_condition and kdu_thread_entity::signal_condition.
