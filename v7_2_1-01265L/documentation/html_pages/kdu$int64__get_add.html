kdu_interlocked_int64::get_add

kdu_int64 get_add( kdu_int64 val)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Add val to the variable and return the original value of the variable (prior to addition), without any memory barriers or any guarantee of atomicity in the addition. This function is provided ONLY FOR SAFE CONTEXTS in which only one thread can possibly be interacting with the variable. Otherwise, use exchange_add, which offers the same functionality but is performed atomically with appropriate memory barriers.

Arguments

val [kdu_int64]
