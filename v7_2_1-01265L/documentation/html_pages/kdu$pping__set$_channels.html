Synopsis

Convenience function for allocating the source_components and palette arrays required to hold the channel properties. Also sets the num_channels member. Copies any existing channel data, so that you can build up a channel description progressively by calling this function multiple times. Initializes all new source_components entries to -1, all new palette entries to NULL, all new default_rendering_precision entries to 8 and all new default_rendering_signed entries to false.

num [int]
