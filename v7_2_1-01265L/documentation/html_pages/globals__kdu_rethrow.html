::kdu_rethrow

Java: static Kdu_global.Kdu_rethrow

static void kdu_rethrow( kdu_exception exc)

Java: public native static void Kdu_rethrow( int exc)

[Declared in "../coresys/common/kdu_ubiquitous.h"]

Synopsis

You should ideally use this function whenever you need to rethrow an exception of type kdu_exception; at a minimum, this will cause exceptions of type KDU_MEMORY_EXCEPTION to be rethrown as std::bad_alloc which will help maintain consistency within your application when exceptions are converted and passed across programming interfaces as kdu_exception. It also provides a path to future catching and conversion of a wider range of exception types.

exc [kdu_exception]
