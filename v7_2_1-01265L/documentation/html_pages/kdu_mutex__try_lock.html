kdu_mutex::try_lock

bool try_lock( )

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Same as lock, except that the call is non-blocking. If the mutex is already locked by another thread, the function returns false immediately.

Return Value

False if the mutex is currently locked by another thread, or has not been successfully created.
