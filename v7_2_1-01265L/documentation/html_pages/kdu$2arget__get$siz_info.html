Synopsis

The serve target is expected to maintain summary information about each of its codestreams, so that this information can be obtained without having to physically open the codestream every time. The various parameters should be fairly self-explanatory. If component_subs is non-NULL, the value of num_components on entry is taken to hold the size of this array and hence the maximum number of component sub-sampling factors to fill out. Similarly, if output_component_subs is non-NULL, the value of the num_output_components argument on entry is taken to hold the size of this array. Otherwise, the values of these arguments on entry is irrelevant. In any case, the function fills out actual values for all the scalar by-reference arguments.

Return Value

Returns false if the requested codestream is not available.
