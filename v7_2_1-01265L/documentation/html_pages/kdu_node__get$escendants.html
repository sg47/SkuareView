Synopsis

Returns the total number of nodes which may be accessed from this node by recursively executing the access_child function. This information is useful in pre-allocating storage to hold structural information prior to creating compression/decompression machinery. Note that the returned count does not include the current node; for leaf nodes (subbands), therefore, the returned value will always be 0.

Arguments

num_leaf_descendants [int &]

This argument is used to return the total number of descendant nodes which are leaves (subbands). The count does not include the current node, even if it is a leaf node.
