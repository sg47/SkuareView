jpb_target::get_next_timecode

kdu_uint32 get_next_timecode( )

Java: public native long Get_next_timecode( )

[Declared in "../apps/compressed_io/jpb.h"]

Go to class description.

Synopsis

Returns the time code for the next frame that would be written; this is useful if you need to find the initial_timecode value to pass to open when opening a new instance of this object (e.g., to represent frames with a different colour space, maximum bit-rate, etc.).
