jp2_source::access_channels

Java: Jp2_source.Access_channels

jp2_channels access_channels( )

Java: public native Jp2_channels Access_channels( )

[Declared in "../apps/compressed_io/jp2.h"]

Go to class description.

Synopsis

Returns an object which may be used to access information recorded in the JP2 "Component Mapping" and "Channel Definition" boxes. The information from both boxes is merged into a uniform set of channel mapping rules, accessed through the returned jp2_channels object.

Returns an empty interface until the conditions required for successful completion of read_header have been met.
