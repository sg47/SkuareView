jpx_roi::clip_region

Java: Jpx_roi.Clip_region

void clip_region( )

Java: public native void Clip_region( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

This function clips any vertices whose coordinates exceed the range 0 to 2^31-2. It workds correctly on any rectangular or quadrilateral region, as well as elliptical regions. In the latter case, clipping is achieved by adjusting the bounding box and scaling any elliptical_skew parameters.
