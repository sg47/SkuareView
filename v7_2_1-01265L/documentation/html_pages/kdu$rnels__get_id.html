kdu_kernels::get_id

int get_id( )

[Declared in "../coresys/common/kdu_kernels.h"]

Go to class description.

Synopsis

Identifies the particular set of wavelet kernels which are being used, returning -1 if the wavelet kernel does not have its own unique identifier — in that case, the second form of the constructor was used to install the kernels.

Return Value

One of Ckernels_W5X3 or Ckernels_W9X7, unless the first form of the init function was not used, in which case the function returns Ckernels_ATK (i.e., -1).
