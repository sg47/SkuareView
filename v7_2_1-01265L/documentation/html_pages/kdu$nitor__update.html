kdu_thread_dependency_monitor::update

virtual void update( int new_dependencies, int delta_max_dependencies, kdu_thread_entity * caller)

[Declared in "../coresys/common/kdu_threads.h"]

Go to class description.

Synopsis

When this object is attached to a kdu_thread_queue object using kdu_thread_queue::set_dependency_monitor, the present function is invoked with exactly the same arguments as any parent queue's update_dependencies function would be invoked if there were a parent.

This is a pure virtual member function; it must be implemented in a derived class.
