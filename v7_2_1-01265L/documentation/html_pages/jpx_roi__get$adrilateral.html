Synopsis

Returns true if the region of interest is a quadrilateral (including a simple rectangle), setting v1 through v4 to the four vertices, in clockwise order, starting from the top-most vertex. Note that the organization of v1 through v4 is not necessarily unique, because there could be multiple top-most vertices or the vertices could even be co-linear, in which case the quadrilateral is a single straight line with no interior, so there is no distinction between clockwise and anti-clockwise orderings. None of these conditions should pose any particular problem. For simple rectangular regions, v1 is the top-left corner.
