kdu_window::create_context_expansion

Go to class description.

Synopsis

This function creates a new internally managed kdu_range_set object and associates it with a particular kdu_sampled_range element within the contexts object. That element is the one with index which. It may be recovered by invoking contexts.get_range with which as the argument. Once the function returns, the caller may add the indices of the codestreams associated with the relevant contexts using kdu_range_set::add.

Arguments

which [int]
