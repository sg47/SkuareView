kdu_serve_target::get_metatree

virtual const kds_metagroup * get_metatree( )

[Declared in "../apps/kdu_server/kdu_serve.h"]

Go to class description.

Synopsis

Returns the metadata partition tree. The tree must be complete; it is not permitted to grow later. If the target consists of one or more raw code-streams without any metadata at all, the function should still return a non-NULL pointer to a tree with exactly one kds_metagroup object, whose length is 0.
