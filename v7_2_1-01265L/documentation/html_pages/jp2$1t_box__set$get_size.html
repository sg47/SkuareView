Synopsis

Implements kdu_compresssed_target::set_target_size, flushing any buffered box contents immediately after writing the header, using the indicated length. This function has no impact on boxes that were opened as header-less (see open).

Arguments

num_bytes [kdu_long]

Total number of bytes in the box contents (not including the header). Some or all of these may already have been written. If the box has been assigned a rubber length, so that the header has already been written, this function has no effect.
