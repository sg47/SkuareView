kdu_synthesis

Java: class Kdu_synthesis

Derives from kdu_pull_ifc

[Declared in "../coresys/common/kdu_sample_processing.h"]

Synopsis

Implements the subband synthesis processes associated with a single DWT node (see kdu_node). A complete DWT synthesis tree is built from a collection of these objects, each containing a reference to the next stage. The complete DWT tree and all required kdu_decoder objects may be created by a single call to the constructor, kdu_synthesis::kdu_synthesis.
