jpx_layer_target::access_resolution

Java: Jpx_layer_target.Access_resolution

jp2_resolution access_resolution( )

Java: public native Jp2_resolution Access_resolution( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Provides an interface which may be used to specify the aspect ratio and physical resolution of the resolution canvas grid for a single codestream used by this compositing layer, or (if the layer uses multiple codestreams) of the compositing layer reference grid.

It is NOT NECESSARY to access or initialize the jp2_resolution object.
