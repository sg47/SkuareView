kdu_interlocked_ptr [struct]

[Declared in "../coresys/common/kdu_elementary.h"]

Synopsis

This object has the size of a single address (or pointer), but offers in-line functions to support atomic read-modify-write style operations for efficient inter-thread communication. All operations are accompanied by appropriate memory barriers, unless otherwise indicated.
