Synopsis

This function plays the same role for codestream header data-bins as get_meta_instructions does for meta data-bins. The function searches for cache model instructions (atomic and non-atomic) which apply to the indicated code-stream main or tile header data-bin (main header if tnum is -ve), writing the results of such a search to buf and returning the number of written entries.

Return Value

0 if no matching cache model instructions were found; otherwise, the function returns the number of values written to buf, which will never exceed 2.

Arguments

stream_idx [int]

Identifies the codestream of interest; must be non-negative.

tnum [int]

Used to specify a specific tile number (or -1 for the codestream main header).

buf[] [int]

Array with at least 2 entries, to which data is written by the function. The interpretation of these values (N and P) is identical to that described for get_meta_instructions.
