jpx_layer_source::access_channels

Java: Jpx_layer_source.Access_channels

jp2_channels access_channels( )

Java: public native Jp2_channels Access_channels( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns an object which may be used to access information recorded in relevant "Component Mapping" and "Channel Definition" boxes. The information from such boxes is merged into a uniform set of channel mapping rules, accessed through the returned jp2_channels object.
