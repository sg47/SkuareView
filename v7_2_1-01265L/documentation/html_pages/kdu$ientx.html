kdu_clientx

Java: class Kdu_clientx

Derives from kdu_client_translator

[Declared in "../apps/compressed_io/kdu_clientx.h"]

Synopsis

Provides an implementation of the kdu_client_translator interface for the specific case of JP2/JPX files. It is not strictly necessary to provide any translator for JP2 files, but since JP2 is a subset of JPX, there is no problem providing the implementation here. All the translation work is done inside the update member. If it finds the data source not to be JPX-compatible, all translation calls will fail, producing null/false results, as appropriate.
