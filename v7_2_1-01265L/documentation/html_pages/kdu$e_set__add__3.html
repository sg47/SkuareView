Synopsis

Adds a contiguous range of indices to the set. Equivalent to calling the first form of the add function with a kdu_sampled_range object whose kdu_sampled_range::from and kdu_sampled_range::to members are identical to the from and to arguments, with kdu_sampled_range::step set to 1 and kdu_sampled_range::context_type set to 0.
