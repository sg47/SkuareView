Go to class description.

Synopsis

Clears any scribble points which have been added using add_scribble_point. If the set of scribble points is non-empty when this function is called, the current state of the object is pushed onto the undo history, which means that you can get the previous scribble points back by executing a call to undo.

Return Value

A region large enough to include all points and edges affected by the change.
