kdu_thread::set_priority

bool set_priority( int priority)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

Returns true if the threads priority was successfully changed. Use get_priority to find the current priority and a reasonable range of priority values to use in calls to this function.

Return Value

False if the thread has not been successfully created, or the priority value is invalid, or the calling thread does not have permission to set the priority of another thread.

Arguments

priority [int]
