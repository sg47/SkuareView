Synopsis

Returns the index of the last instruction with index strictly less than lim_inst_idx (unless lim_inst_idx <= 0, in which case there is no upper bound) such that the instruction uses the indicated compositing layer. If no such instruction can be found, the function returns -1.

Arguments

layer_idx [int]

lim_inst_idx [int]
