jp2_input_box::get_jpx_box

Java: Jp2_input_box.Get_jpx_box

virtual jpx_input_box * get_jpx_box( )

Java: public native Jpx_input_box Get_jpx_box( )

[Declared in "../apps/compressed_io/jp2.h"]

Go to class description.

Synopsis

This function exists to facilitate the transporting of references to jpx_input_box objects across function interfaces that accept references to the base jp2_input_box object, where the implementation of the function interface may benefit from or require the enhanced feature set of the jpx_input_box class.

One important application is the passing of jpx_input_box objects across the kdu_compressed_video_source::open_stream interface. If the implementation of that function actually needs to invoke jpx_codestream_source::open_stream it will need a reference to a jpx_input_box so that all types of codestream sources can be opened and handled as frames of an abstract video sequence. However, there is nothing about the virtual base function jpx_codestream_source::open_stream that necessitates the passing of anything more than a jp2_input_box reference.

The base implementation of this function returns NULL — meaning that jp2_input_box objects cannot be treated as jpx_input_box objects. However, of course, jpx_input_box overrides this virtual function to return itself. Essentially, this function just does dynamic casting in a very specific way, and one that is portable across language interfaces.
