kdu_image_dims::add_component

void add_component( int height, int width, int bit_depth, bool is_signed, int comp_idx=-1)

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

If comp_idx is < 0, this function appends a new component to the evolving description. Otherwise, the function appends as many components as required to ensure that the component with index comp_idx exists, and then writes or overwrites its contents with the supplied values. Where new components must be created, the append_component function is used. That function replicates the properties of any previous components for which information is available.

Arguments

height [int]

width [int]

bit_depth [int]

is_signed [bool]
