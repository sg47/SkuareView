jpx_source::access_compatibility

Java: Jpx_source.Access_compatibility

jpx_compatibility access_compatibility( )

Java: public native Jpx_compatibility Access_compatibility( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns an interface which may be used to discover whether or not the data source is compatible with JP2 or baseline JPX, and also to discover the set of features which have been used (as recorded in a reader requirements box) and which features are considered essential for decoding. Returns an empty interface if the open function has not yet returned successfully.
