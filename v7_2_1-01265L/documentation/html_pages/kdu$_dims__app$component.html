kdu_image_dims::append_component

void append_component( )

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

This function adds a new component, replicating its dimensions and precision information, based on the last available component — if there is none, all parameters for the new component are set to 0. The parameters may later be overwritten using the add_component function.
