::jp2_4cc_to_string

const char * jp2_4cc_to_string( kdu_uint32 box_type, char buf[])

[Declared in "../apps/compressed_io/jp2.h"]

Synopsis

Converts box_type to a 4-character string, which is written to the supplied buf array. The buf array must be able to hold at least 5 characters (including the null terminator). As a matter of convenience, the function also returns a pointer to the buf array.

The box_type argument is interpreted as a four-character code, whose first character appears in the MSB and whose last character appears in the LSB. Usually, the four bytes of the box_type value are printable ASCII characters. To ensure that the string consists of printable ASCII characters, the following conversions are applied:

  • Spaces are converted to underscore characters '_'
  • Characters with codes less than 0x20 or greater than 0x7F are converted to '.'.

Arguments

box_type [kdu_uint32]

buf[] [char]
