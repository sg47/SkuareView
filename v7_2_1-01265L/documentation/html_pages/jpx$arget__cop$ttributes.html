Go to class description.

Synopsis

Copies the jp2_dimensions and jp2_palette information from src to the current object. This avoids the need to call access_dimensions and access_palette and initialize the contents of the returned interfaces. However, you still need to write the JPEG2000 codestreams themselves using open_stream or add_fragment and write_fragment_table.

src [jpx_codestream_source]
