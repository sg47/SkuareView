Synopsis

Same as the first form of get_frame_instant, but returns the frame starting time for a specific frame, identified by frame_idx. The result should be the same as that obtained by first seeking to frame_idx and then invoking the first form of the function, but saves the need to change the current frame context by invoking seek_to_frame.

If the indicated frame does not exist, the function returns 0.

Arguments

frame_idx [int]
