jp2_output_box::get_header_length

Synopsis

Returns the number of bytes in the box header. If this function is called before all bytes in the box have been written, the returned length may be too small, sice boxes with size greater than or equal to 2^32 require 16-byte headers instead of 8-byte headers. Usually, you will wait until you are about to close he box (or even after you have closed the box) before calling this function.

If you need to be sure of the header length ahead of time, you can invoke use_long_header, which forces the header to be written with an 8-byte length field, rather than a 4-byte length field, regardless of its actual length.
