kdu_sample_allocator::alloc32

kdu_sample32 * alloc32( int before, int after, size_t alloc_off, int inst=0)

[Declared in "../coresys/common/kdu_sample_processing.h"]

Go to class description.

Synopsis

Same as alloc16, except that it allocates storage for arrays of 32-bit quantities. The alloc_off value supplied here must have been obtained from a call to pre_alloc in which the use_shorts argument was false.

Arguments

before [int]

after [int]

alloc_off [size_t]
