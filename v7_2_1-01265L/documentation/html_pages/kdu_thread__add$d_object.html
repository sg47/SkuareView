kdu_thread::add_thread_object

bool add_thread_object( kdu_thread_object * obj)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

You can use this function to add obj to an internal list of objects that will be deleted when the thread returns from its entry-point function.

If the check_self function returns false, or kdu_thread::create was not used to create the thread of execution, this function does nothing, returning false.

The function returns true if obj was already on the list of if the function added it to the list; in either case, responsibility for deleting obj has been transferred to the kdu_thread object.

Arguments

obj [kdu_thread_object *]
