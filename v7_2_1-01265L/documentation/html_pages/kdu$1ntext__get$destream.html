Synopsis

See kdu_serve_target::access_context for a general introduction. The member_idx argument is used to enumerate the individual codestreams which belong to the context. It runs from 0 to the value returned by get_num_members less 1.

Return Value

The index of the codestream associated with the indicated member of the codestream context.
