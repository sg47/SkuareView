[Declared in "../apps/compressed_io/jpb.h"]

Go to class description.

Synopsis

Returns the number of ticks associated with the frame to which the currently open image belongs. If no image is currently open, the function returns the frame period associated with the frame to which the next open image would belong if open_image were called.

This function returns 0 if no frame-rate information is available for the frame in question or if there is no open image and no further images can be opened.

Note that if the video is interlaced, there are two images (fields) in each frame period.
