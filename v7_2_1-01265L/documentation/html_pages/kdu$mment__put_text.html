Synopsis

Returns false if the comment contents have been frozen, as a result of writing it to or converting it from a code-stream comment marker segment. Also returns false if the put_data function has already been used to store binary data in the object, as opposed to text. Otherwise, appends the text in string to any existing text and returns true. The string should be NULL-terminated ASCII although nothing explicitly prevents you from embedding arbitrary UTF-8 strings in codestream comments.
