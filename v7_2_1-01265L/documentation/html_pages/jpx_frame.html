jpx_frame

Java: class Jpx_frame

[Declared in "../apps/compressed_io/jpx.h"]

Synopsis

Manages the information associated with a single composited display frame, as determined from information embedded within a JPX file's top-level Composition box, or within JPX containers (Compositing Layer Extensions boxes). This interface is usable only when reading an existing file (possibly sourced from a dynamic cache — i.e., a kdu_cache object) — it is not of interest to file writers.

Objects of this class are merely interfaces to an internal object, which cannot be directly created by an application. Use jpx_composition::access_frame to obtain a non-empty interface.
