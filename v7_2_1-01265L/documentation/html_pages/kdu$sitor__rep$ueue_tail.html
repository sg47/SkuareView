Synopsis

This function is identical to push_composition_buffer, except that the most recently pushed frame buffer is removed from the queue and replaced with this one. This can be very useful if a newly generated composition buffer is found to obviate the need from the most recent one, but we do not want to pop all buffers from the head of the queue.
