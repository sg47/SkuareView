kdu_semaphore::create

bool create( int initial_value)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

You must explicitly call create and destroy. The constructor and destructor for the kdu_semaphore object do not create or destroy the underlying synchronization primitive itself.

Return Value

False if semaphore creation is not supported on the current platform or if the operating system is unwilling to allocate any more synchronization primitives to the process. The former problem may be resolved by creating the appropriate definitions in "kdu_elementary.h".

Arguments

initial_value [int]

Initial value for the semaphore. Calls to wait attempt to decrement the count, returning only when the result of such decrementing would be non-negative.
