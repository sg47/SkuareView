kdu_image_dims::get_signed

bool get_signed( int comp_idx)

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

Returns true if the indicated image component holds signed information, with a nominal range from -2^{B-1} to +2^{B-1}-1, where B is the bit-depth returned by get_bit_depth. Otherwise, the function returns false and the nominal data range is from 0 to 2^B.

Arguments

comp_idx [int]
