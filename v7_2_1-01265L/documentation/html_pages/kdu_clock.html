kdu_clock

[Declared in "../coresys/common/kdu_elementary.h"]

Synopsis

This class is provided primarily as a convenience for timing the execution of programs or parts of programs. The intent is to overcome a weakness in the implementation of the ANSII C clock function, in that this function returns ellapsed CPU time on some operating systems and ellapsed system clock time on others. For a more comprehensive timing solution, you might be interested in the kdcs_timer class defined as part of Kakadu's communication support system. The present object, however, is designed to be extremely simple and implemented entirely in-line, as with everything in the "kdu_elementary.h" header.
