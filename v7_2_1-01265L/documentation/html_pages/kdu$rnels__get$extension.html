kdu_kernels::get_symmetric_extension

bool get_symmetric_extension( )

[Declared in "../coresys/common/kdu_kernels.h"]

Go to class description.

Synopsis

Returns true if the symmetric boundary extension policy is to be used at image boundaries within each lifting step. The centre of symmetry in this case is the boundary sample location within an interleaved representation in which low-pass samples occupy the even indexed locations and high-pass samples occupy the odd indexed locations.

If the function returns false, the sample-replication is to be used at the boundaries within each lifting step.
