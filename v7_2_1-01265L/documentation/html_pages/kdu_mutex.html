kdu_mutex

[Declared in "../coresys/common/kdu_elementary.h"]

Synopsis

The kdu_mutex object implements a conventional mutex (i.e., a mutual exclusion synchronization object), on Windows, Unix/Linux and MAC operating systems at least — basically, any operating system which supports the "pthreads" or Windows threads interfaces, assuming the necessary definitions are set up in "kdu_elementary.h".

From Kakadu version 7.0, the create function offers some optional customization features which may result in the instantiation of different types of locking primitives, depending on the platform.
