kdu_resolution::which

int which( )

Java: public native int Which( )

[Declared in "../coresys/common/kdu_compressed.h"]

Go to class description.

Synopsis

Gets the resolution level index. This is the same index which must be supplied to kdu_tile_comp::access_resolution in order to obtain the present interface. 0 is the index of the resolution level which contains only the LL subband.
