Synopsis

Returns false if the comment contents have been frozen, as a result of writing it to or converting it from a code-stream comment marker segment. Also returns false if the put_text function has already been used to store text in the object, as opposed to binary data. Otherwise, appends the num_bytes of data supplied in the data array to any data previously supplied, and returns true.
