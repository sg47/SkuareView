kdu_semaphore::wait

bool wait( )

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

This function is used to wait for the semaphore to become positive, whereupon it is atomically decremented by 1 and the function returns.

Return Value

False if the semaphore has not been successfully created
