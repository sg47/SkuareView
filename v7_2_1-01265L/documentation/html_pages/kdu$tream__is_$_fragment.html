kdu_codestream::is_last_fragment

Java: Kdu_codestream.Is_last_fragment

bool is_last_fragment( )

Java: public native boolean Is_last_fragment( )

[Declared in "../coresys/common/kdu_compressed.h"]

Go to class description.

Synopsis

Returns true if the flush function can be expected to write an EOC marker. This should be the case whenever the codestream was created for output (it will never be the case if it was created for input or interchange), except when compressing a non-terminal fragment.
