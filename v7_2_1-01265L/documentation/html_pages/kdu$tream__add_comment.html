kdu_codestream::add_comment

Java: Kdu_codestream.Add_comment

kdu_codestream_comment add_comment( )

Java: public native Kdu_codestream_comment Add_comment( )

[Declared in "../coresys/common/kdu_compressed.h"]

Go to class description.

Synopsis

Returns an interface to a new comment, to which text may be written. Returns an empty interface unless the code-stream management machinery was created for output or for interchange, and flush has not yet been called.

This function is not currently thread-safe.
