jp2_palette::get_num_entries

[Declared in "../apps/compressed_io/jp2.h"]

Go to class description.

Synopsis

The input component samples are to be clipped to the range 0 through entries-1, where entries is the value returned by this function. All lookup tables have exactly the same number of entries.

Return Value

0 if no palette is being used.
