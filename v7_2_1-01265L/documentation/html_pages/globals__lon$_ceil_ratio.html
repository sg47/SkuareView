[Declared in "../coresys/common/kdu_utils.h"]

Synopsis

Returns the ceiling function of the ratio num / den, where the denominator is required to be strictly positive. The result must fit within a signed 32-bit integer, even if the numerator or denominator do not.

Return Value

Non-negative ratio.

Arguments

num [kdu_long]

Non-negative numerator.

den [kdu_long]

Non-negative denomenator.
