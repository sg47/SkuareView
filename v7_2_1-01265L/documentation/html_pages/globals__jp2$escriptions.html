Synopsis

This function adds explicit textualization functions to the textualizer object, which are capable of generating XML-compatible textual descriptions of the relevant box contents. However, this is only done for some of the basic box types used by JP2 files. You can augment the set of boxes for which descriptions and textualization functions are available by updating the initial machinery installed by this function through further calls to textualizer.add_box_type. This is typically done by calling functions like jpx_add_box_descriptions, mj2_add_box_descriptions and jpb_add_box_descriptions.
