jpx_frame_expander::get_num_members

Java: Jpx_frame_expander.Get_num_members

int get_num_members( )

Java: public native int Get_num_members( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns the number of members which were installed by the most recent call to construct. Details of these members may be recovered using the get_member function. If the most recent call to construct requested a non-existent frame, the value returned by the present function will be 0.
