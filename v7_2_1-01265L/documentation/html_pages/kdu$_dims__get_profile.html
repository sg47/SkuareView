kdu_image_dims::get_profile

const kdu_byte * get_profile( )

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

Returns a buffer containing the ICC profile whose length is identified by the last argument to get_colour_info; returns NULL if the length of that buffer was not at least 4 — the number of bytes required to embed the length of the profile. The first 4 bytes of the returned buffer contains a big-endian representation of the length which was returned via get_colour_info.
