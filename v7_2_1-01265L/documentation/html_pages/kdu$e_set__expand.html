Synopsis

Writes all unique indices which fall within the ranges defined in this set, while simultaneously lying within the interval [ accept_min, accept_max] into the supplied buffer, returning the number of written indices. It is your responsibility to ensure that buf is large enough to accept all required indices, which can easily be done by ensuring at least that buf can hold accept_max+1- accept_min entries.
