Synopsis

Use this function to navigate to the next instance, if any, in the current object's instance list. If there are no more instances, the function returns NULL. This is useful primarily for derived parameter classes which treat instances like components. In this case, random access to instances is possible and the instance list might be sparse. This function provides an efficient way to navigate instance lists for this type of derived parameter class (an example is the ads_params class).
