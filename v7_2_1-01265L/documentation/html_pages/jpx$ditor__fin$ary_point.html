Synopsis

Similar to find_nearest_anchor, except that the function looks for the nearest location to point which lies on a region boundary. To do this, the function projects point onto each boundary in turn and measures the distance to the projection. On exit, the function returns with the nearest boundary location in point, unless there are none.

Return Value

True if any projected points exist, from which the nearest can be picked. This condition depends on whether or not the editor is empty, as well as the exclude_selected_region argument.

Arguments

point [kdu_coords &]

exclude_selected_region [bool]

If true, any currently selected region is excluded from the boundary projection search. If there is only one region and it happens to be selected, the function will return false.
