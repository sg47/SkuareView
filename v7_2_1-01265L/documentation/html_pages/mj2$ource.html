mj2_video_source

Java: class Mj2_video_source

Derives from kdu_compressed_video_source

Derives from kdu_compressed_source

[Declared in "../apps/compressed_io/mj2.h"]

Synopsis

Manages access to a single video track within an MJ2 presentation. Use the mj2_source::access_video_track function to access an instance of this class, which may then be used as a compressed data source (on a frame-by-frame basis) in calls to kdu_codestream::create.
