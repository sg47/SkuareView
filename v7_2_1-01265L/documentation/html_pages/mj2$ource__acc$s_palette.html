mj2_video_source::access_palette

Java: Mj2_video_source.Access_palette

jp2_palette access_palette( )

Java: public native Jp2_palette Access_palette( )

[Declared in "../apps/compressed_io/mj2.h"]

Go to class description.

Synopsis

Returns an object which may be used to access any information recorded in the JP2 "Palette" box. If there is none, the object's jp2_palette::get_num_luts function will return 0. This information is fixed for all frames in the video track.
