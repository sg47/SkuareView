kdu_image_dims::get_source_metadata

jp2_output_box * get_source_metadata( int idx)

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

Use this function to retrieve any extra meta-data boxes which have been added by the file format reader. These boxes may be copied into a JP2/JPX file which is under construction, to record the original file's meta-data.

Return Value

NULL if the number of boxes added via add_source_metadata is less than or equal to idx.

Arguments

idx [int]
