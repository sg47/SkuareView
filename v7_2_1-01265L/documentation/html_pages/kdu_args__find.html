kdu_args::find

char * find( const char * pattern)

[Declared in "../apps/args/kdu_args.h"]

Go to class description.

Synopsis

Returns NULL unless an argument matching the supplied pattern string can be found, in which it returns a pointer to the internal copy of this argument string. Currently, only direct string matching on the pattern string is supported.

Note carefully that the returned string points to an internal resource, which might be deleted by the advance function.
