jpx_roi_editor::split_selected_anchor

Java: Jpx_roi_editor.Split_selected_anchor

kdu_dims split_selected_anchor( )

Java: public native Kdu_dims Split_selected_anchor( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

This function does nothing unless select_anchor has been used to select an anchor point and the selected anchor point has the same location as other anchor points. If these conditions are met, the currently selected anchor point is moved slightly so that it can be dragged and moved independently of the others.

Return Value

A region large enough to include all points and edges affected by the change. If there is no change, this region is empty.
