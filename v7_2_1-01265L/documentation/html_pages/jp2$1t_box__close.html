jp2_output_box::close

Java: Jp2_output_box.Close

virtual bool close( )

Java: public native boolean Close( )

Overrides kdu_compressed_target::close

[Declared in "../apps/compressed_io/jp2.h"]

Go to class description.

Synopsis

Flushes the contents of the box to the output. If the box had rubber length, there will be nothing to flush. If the output device is (or was) unable to accept all of the box contents, this function returns false.
