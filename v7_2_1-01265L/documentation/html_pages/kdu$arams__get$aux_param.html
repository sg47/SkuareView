Synopsis

Returns a copy of the nth auxiliary painting parameter supplied via the most recent call to kdu_region_compositor::configure_overlays. Implementations of kdu_region_compositor::paint_overlay or kdu_region_compositor::custom_paint_overlay may use this parameter to adjust the appearance of the overlay image. These auxiliary painting parameters are used extensively by the default kdu_region_compositor::paint_overlay function; moreover, a custom implementation may call the default version of the function after temporarily replacing the auxiliary painting parameters with the aid of push_aux_params.

Arguments

n [int]
