Synopsis

This function may be used to alter the type of memory access which can be performed on the buffer. If read access was not previously allowed, but is now required, memory resources may need to be allocated in a different way.

If the memory manager is unable to retain the contents of a buffer which was originally allocated as write-only, but must now be available for read-write access, it must return false. This informs the caller that the buffer's contents must be marked as invalid so that they will be regenerated by subsequent calls to kdu_region_compositor::process.

Return Value

False if the buffer's contents must be regenerated. This only happens if read_access_required is true and the memory manager cannot preserve the contents of the originally allocated buffer.

Arguments

read_access_required [bool]
