kdu_error::flush

virtual void flush( bool end_of_message)

Overrides kdu_message::flush

[Declared in "../coresys/common/kdu_messaging.h"]

Go to class description.

Synopsis

Invokes the kdu_message::flush function associated with the error message handler supplied to kdu_customize_errors, if not NULL, being careful to always set end_of_message to false in this forwarded call. This ensures that an end_of_message value of true will uniquely identify the fact that the kdu_error object is being destroyed.

Arguments

end_of_message [bool]
