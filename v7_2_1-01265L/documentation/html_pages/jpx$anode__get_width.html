jpx_metanode::get_width

Java: Jpx_metanode.Get_width

int get_width( )

Java: public native int Get_width( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns 0 unless the node represents an ROI description box (i.e., if get_num_regions returns non-zero). In this case, the function returns the width of the widest region described by the ROI description box, as measured by jpx_roi::measure_span.
