kdu_thread_job::set_job_func

void set_job_func( kdu_thread_job_func func)

[Declared in "../coresys/common/kdu_threads.h"]

Go to class description.

Synopsis

Be sure to call this function before passing a reference to this object to the kdu_thread_queue::bind_jobs function. Note that this function also removes the connection between this object and any internal thread queue resource to which it was bound by a prior call to kdu_thread_queue::bind_jobs or an auto-bind call to kdu_thread_queue::schedule_job. It follows that this function should always be called first, before binding.
