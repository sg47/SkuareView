kdu_run_queue_job::init

void init( kdu_thread_entity * caller)

[Declared in "../coresys/common/kdu_threads.h"]

Go to class description.

Synopsis

Call this function before you first enter the job function. You should call this function from the master thread (almost invariably the thread group owner) that is described in the documentation of kdu_run_queue. This function just sets up the initial kdu_thread_entity_condition pointer to reference the calling thread's active condition object so that it can be signalled from within the job function.

Arguments

caller [kdu_thread_entity *]
