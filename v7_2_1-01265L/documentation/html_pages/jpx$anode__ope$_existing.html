jpx_metanode::open_existing

Java: public native boolean Open_existing( Jp2_input_box box)

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

This function uses get_existing to identify the jp2_family_src object and location within that source of the box associated with the current node. If get_existing returns a non-empty location, the function uses it to open the supplied box and return true. Otherwise, the function simply returns false.

box [jp2_input_box &]
