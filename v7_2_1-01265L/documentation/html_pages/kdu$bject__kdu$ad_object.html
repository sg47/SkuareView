kdu_thread_object::kdu_thread_object

kdu_thread_object( const char * name_string=NULL)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

If name_string is NULL (default), the constructed object can be registered with kdu_thread::add_thread_object, but cannot be retrieved via kdu_thread::find_thread_object. This is fine if all you want to do is ensure that the object gets deleted when the thread returns from its entry-point function.

Arguments

name_string [const char *]
