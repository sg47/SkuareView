kdu_image_dims

[Declared in "../apps/image/kdu_image.h"]

Synopsis

Container for dimensions, precisions and signed/unsigned characteristics of one or more image components, along with resolution information, if available. An object of this type must be passed to kdu_image_out::kdu_image_out.

In addition to component dimensions, this object also provides facilities for exchanging display resolution information with the compression/decompression application, as well as methods for exchanging additional meta-data and knowledge (if any) about the colour space. These methods appear first, below.
