Synopsis

This second form of the get_icc_profile function copies at most buf_len bytes of the ICC profile to the supplied buffer and returns the actual length of the buffer. If you just want to know the size of the profile, you can pass NULL for buffer and the buf_len is ignored. This version of the function is safer, since it does not offer access to internal resources; it is suitable for use with foreign language bindings such as Java or C#.
