kdu_image_out( )

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

Constructs an empty interface — one whose exists function returns false. To open an image file, you must either use the second form of the constructor or else assign another, non-empty, kdu_image_out interface to the current one.
