kdu_image_dims::set_meta_manager

void set_meta_manager( jpx_meta_manager meta_manager)

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

You may use this function when writing a file which is decompressed from a JP2/JPX file, to supply the jpx_source object's meta-manager. This allows the file-format writing engine to search for additional meta-data which might be used to construct more meaningful headers. This is most useful when writing to rich image file formats such as TIFF.

Arguments

meta_manager [jpx_meta_manager]
