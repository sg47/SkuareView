kdu_metareq::get_qualifier

Java: Kdu_metareq.Get_qualifier

int get_qualifier( )

Java: public native int Get_qualifier( )

[Declared in "../apps/compressed_io/kdu_client_window.h"]

Go to class description.

Synopsis

Returns the value of the public qualifier member. Useful for non-native language bindings such as Java. Note that the return value is a union of one or more of the flags: KDU_MRQ_ALL, KDU_MRQ_GLOBAL, KDU_MRQ_STREAM and KDU_MRQ_WINDOW.
