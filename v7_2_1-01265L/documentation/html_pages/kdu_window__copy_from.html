kdu_window::copy_from

[Declared in "../apps/compressed_io/kdu_client_window.h"]

Go to class description.

Synopsis

Copies the window represented by src. If copy_expansions is true, the context expansions (see kdu_sampled_range::expansion) within the contexts range set are also copied. Otherwise, expansions are not copied, and kdu_sampled_range::expansion is set to NULL for each copied element of the contexts object.

Arguments

src [const kdu_window &]

copy_expansions [bool]
