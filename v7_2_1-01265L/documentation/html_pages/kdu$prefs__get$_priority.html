Synopsis

Implements the colour description prioritization algorithm described in conjunction with colour_meth_pref_limits. If you have several colour descritions, you can supply the values returned by jp2_colour::get_space, jp2_colour::get_precedence and jp2_colour::get_approximation_level to this function and select the colour description whose returned priority value is smallest.
