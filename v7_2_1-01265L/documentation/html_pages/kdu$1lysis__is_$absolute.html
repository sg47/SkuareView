Synopsis

Returns true if the indicated line has been assigned a reversible (i.e., absolute integer) representation by the create function. Otherwise, calls to exchange_line will return lines whose kdu_line_buf::is_absolute function returns false. This function is provided as a courtesy, so that applications can know ahead of time what the type of the data associated with a line will be. In the presence of multi-component transforms, this can be non-trivial to figure out based solely on the output component index.
