kdu_image_dims::add_source_metadata

jp2_output_box * add_source_metadata( kdu_uint32 box_type)

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

When reading the header of a rich image file format, such as TIFF, additional meta-data may be discovered which could be passed to a JP2/JPX compressed image. This may be done by calling this function to create a new open jp2_output_box into which the reader can write box contents.

Be sure not to invoke jp2_output_box::close, since that will destroy the box's contents.

Arguments

box_type [kdu_uint32]
