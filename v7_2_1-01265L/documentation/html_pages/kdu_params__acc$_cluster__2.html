Synopsis

Returns a particular cluster head by its position in the underlying cluster list. The function may be invoked on any member of any cluster in the list. The first cluster has sequence_idx=0 and may also be retrieved by supplying a NULL cluster_name argument. This first cluster will generally correspond to the siz_params class.

Return Value

NULL if the requested cluster does not exist.

Arguments

sequence_idx [int]

0 for the first cluster in sequence.
