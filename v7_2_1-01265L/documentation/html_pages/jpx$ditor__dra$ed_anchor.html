Synopsis

Modifies the internal state of the object to reflect the potential effects of moving the selected anchor point to new_point, so that calls to get_edge and get_anchor which specify dragging=true will retrieve information about the locations of the anchor point and edges affected by such a potential move operation.

Return Value

A region large enough to include all points and edges affected by the change — this includes existing drag vertices and edges and new drag vertices and edges. If there is no change, this region is empty.

Arguments

new_point [kdu_coords]
