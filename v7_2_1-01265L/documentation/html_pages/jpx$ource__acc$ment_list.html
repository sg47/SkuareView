jpx_codestream_source::access_fragment_list

Java: Jpx_codestream_source.Access_fragment_list

jpx_fragment_list access_fragment_list( )

Java: public native Jpx_fragment_list Access_fragment_list( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

This function returns a non-empty fragment list interface if the codestream is represented by a fragment table box and the embedded fragment list box's contents are fully available. Otherwise it returns an empty interface. Equivalently, a non-empty interface is returned if the codestream is ultimately represented by a fragment table and stream_ready returns true.

The returned fragment list may be passed to jpx_input_box::open to open the codestream to which the fragments point, as though it were a contiguous codestream.
