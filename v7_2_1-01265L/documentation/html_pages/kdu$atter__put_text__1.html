Synopsis

Implements the kdu_message::put_text function, collecting text as it appears and formatting it into lines with appropriate indentation and line breaks.

Note that the current implementation supports only ASCII text formatting (at least it assumes that each character corresponds to a single byte). If you want to support Unicode or multi-byte UTF-8 strings, you are best off implementing your own kdu_message-derived object from scratch, as explained in the introductory comments to this kdu_message_formatter class.
