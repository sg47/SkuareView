kdu_kernels::init

Overload navigation: 1, 2

void init( int kernel_id, bool reversible)

[Declared in "../coresys/common/kdu_kernels.h"]

Go to class description.

Synopsis

Initialize the object to work with one of the two standard DWT kernels defined by Part-1 of the JPEG2000 standard. For more general kernels, use the second form of the init function.

If the object was already initialized, it is first reset to the empty state before re-initialization.

Arguments

kernel_id [int]

Must be one of Ckernels_W5X3 or Ckernels_W9X7.

reversible [bool]

Only affects subband normalization gains. The gains are both set to 0 if reversible is true — see get_lifting_factors.
