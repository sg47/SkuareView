kdu_window_context

Java: class Kdu_window_context

[Declared in "../apps/compressed_io/kdu_client_window.h"]

Synopsis

This interface is used to query and translate JPIP contexts.

Objects of this class are merely interfaces to an internal object managed by kdu_serve_target or kdu_client_translator. One of the functions, kdu_serve_target::access_context or kdu_client_translator::access_context, implemented in an appropriate derived class, is used to obtain a non-empty interface.
