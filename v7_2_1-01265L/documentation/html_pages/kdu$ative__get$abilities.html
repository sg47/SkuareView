Synopsis

Implements kdu_compressed_source::get_capabilities again so as to provide a callback binding for implementation in a foreign language derived class. The implementing class must provide its own overriding definition for this function, which returns one of: KDU_SOURCE_CAP_SEQUENTIAL, KDU_SOURCE_CAP_SEEKABLE, KDU_SOURCE_CAP_CACHED or KDU_SOURCE_CAP_IN_MEMORY. See the description of kdu_compressed_source::get_capabilities for an explanation of these constants.
